# LinguaChat — User Manual & Design Guide

Welcome to **LinguaChat**, a state-of-the-art real-time multilingual messaging application. This document details how the application works, how to use its core features, and provides explanations for its custom theme system and advanced chat controls.

---

## 1. Introduction & Core Features

LinguaChat breaks language barriers by automatically translating messages server-side before they reach your screen. You type in your preferred language, and everyone in the chat room understands in theirs instantly.

### Key Features:
* **Zero-Translation Delay**: Server-side translation using mock translators.
* **12 Premium Workspace Themes**: Fully customizable spaces, including 7 creative themes with translucent panels, backdrop blur, and background images.
* **Time-Limited Messages**: Edit or unsend any sent message within a 2-minute window.
* **Privacy First**: End-to-end socket level encryption using dynamic room keys.

---

## 2. Getting Started: Step-by-Step

### Step 1: Login & Setup
1. **Enter Username**: Type a unique display name (max 15 characters).
2. **Select Your Language**: Choose from 11 supported languages:
   * 🇺🇸 English (`en`)
   * 🇮🇳 Hindi (`hi`)
   * 🇷🇺 Russian (`ru`)
   * 🇪🇸 Spanish (`es`)
   * 🇫🇷 French (`fr`)
   * 🇩🇪 German (`de`)
   * 🇨🇳 Chinese (`zh`)
   * 🇸🇦 Arabic (`ar`)
   * 🇵🇹 Portuguese (`pt`)
   * 🇯🇵 Japanese (`ja`)
   * 🇹🇷 Turkish (`tr`)
3. **Choose Avatar**: Enter a text character or select an emoji avatar.
4. **Choose Action**:
   * **Create Room**: Click this to host a new chat session.
   * **Join Room**: Enter a 6-digit code and click this to join an existing session.

### Step 2: The Room Lobby
If you create a room, you enter the **Room Lobby**.
* Here, you can copy the 6-digit **Room Code** to share with others.
* View connected participants in the list.
* Click **Start Chat** once everyone has joined.

---

## 3. Using the Chat Room

### Real-Time Translation
* Simply type your message in your language and hit **Send**.
* Recipients will receive the message translated into their selected language automatically.
* A globe icon (`🌐`) will appear on translated messages. Click **Show Original** to view the sender's original untranslated text.

### Time-Limited Edit & Unsend (2-Minute Rule)
To ensure conversation integrity while giving users flexibility:
* Hover (on Desktop) or tap (on Mobile) on your message bubble within **2 minutes** of sending.
* Click **Edit (✏️)** to update your message. The input bar will display an "Editing message..." banner. Click **Save** to push updates, which will automatically re-translate and update for all recipients.
* Click **Unsend (🗑️)** to permanently delete the message from the server and remove it from everyone's screen.
